MrSplitMan
==========

Weekend Project IRC Bot in Erlang. This is not serious, I am using this
project to teach myself Erlang.

Requirements
------------
 - Erlang OTP R14
 - Make

Compilation/Startup
-------------------
You need to compile before running it::

    make
    make start
